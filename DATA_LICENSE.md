# Dataset Information and Attribution

## Data Source

This project uses the **DLD Transactions (Open)** dataset published by the **Dubai Data & Statistics Establishment (DDSE)** through the Dubai Open Data platform.

Official source:

- https://data.dubai/en/l/470061

## Ownership

The dataset is owned and maintained by the Government of Dubai through the Dubai Data & Statistics Establishment (DDSE).

## Dataset Distribution

The raw dataset is not included in this repository.

Users wishing to reproduce this project should obtain the data directly from the official source listed above.

## Repository Contents

This repository contains:

- Source code
- Jupyter notebooks
- Documentation
- Visualizations
- Derived analysis
- Machine learning models
